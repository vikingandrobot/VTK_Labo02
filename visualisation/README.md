# Visualisation

Ce dossier contient le(s) programme(s) qui permettent de visualiser les solutions du casse-tête. Il s'agit pour l'instant d'une première version tout à fait bancale.

Le fichier [solution.txt](.solution.txt) contient un exemple de format à respecter pour les solutions.

## Visualisation de solution multi-vues

Pour lancer le programme de visualisation multi-vue, placer la solution dans le fichier solution.txt, 

et lancer la commande 

```
python3 solution_multivue.py
```
